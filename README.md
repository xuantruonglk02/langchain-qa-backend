# LangChain Q&A Backend

## Installation

1. Clone this repository

    ```bash
    git clone git@github.com:xuantruonglk02/langchain-qa-backend.git
    ```

2. Install dependencies
    ```bash
    npm install
    ```

## Run this app

-   On dev mode
    ```bash
    npm run dev
    ```
-   On production mode
    ```bash
    npm run build
    npm start
    ```
